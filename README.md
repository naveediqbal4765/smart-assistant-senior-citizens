## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React.js + Tailwind CSS |
| Backend | Node.js + Express.js |
| Database | MongoDB Atlas |
| Real-time | Socket.io |
| Auth | JWT + bcryptjs |
| Email | Nodemailer |

---

## 🚀 Quick Start

```bash
# Clone repo
git clone https://github.com/naveediqbal4765/smart-assistant-senior-citizens.git
cd smart-assistant-senior-citizens

# Setup Backend
cd backend && npm install && cp .env.example .env
# Edit .env with your credentials
npm run dev

# Setup Frontend (new terminal)
cd ../frontend && npm install && npm start
```

📖 See [DOCUMENTATION.md](./DOCUMENTATION.md) for full setup guide.

- **Elder** - Main user: gets help, health monitoring, SOS alerts
- **Caregiver** - Monitors elder's health data and location remotely  
- **Volunteer** - Community member who helps elders with tasks
